Thinking about Machine learning without maths and statistics is not possible. Machine learning is not at all about implementing the algorithms over the datasets and generating results, it’s all about how do you understand the stats and insights of data so that you can apply the algorithms in a better way. At the end of this course, one will be able to understand the maths behind the machine learning algorithms and apply them to real world datasets. The complete course will be hands-on plus theory.
The pre-requisite for this course is Python as all hands-on sessions will be in python.
